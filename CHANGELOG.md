# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-24

### Added
- DNS management commands (create, list, delete)
- Caddy reverse proxy configuration
- R2 storage integration
- Service health check functionality
- AI agent mode with JSON Lines output
- Pebble Spec v1.1 compliance
- Environment variable configuration support
- Comprehensive error handling

### Security
- All sensitive credentials moved to `.env` file
- Example configuration template (`.env.example`)
- No hardcoded API keys or domain names

### Documentation
- Bilingual documentation (EN/CN)
- Detailed usage examples
- API reference guide
